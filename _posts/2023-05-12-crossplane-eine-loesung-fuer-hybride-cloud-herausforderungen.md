Crossplane ist ein plattformübergreifendes Kontrollsystem (Control-Plane), das das Management von Cloud-Ressourcen vereinfachen und automatisieren soll. Es ermöglicht die Verwaltung verschiedener Cloud-Provider und lokaler Ressourcen unter einer einheitlichen API und kann so Komplexität in hybriden Umgebungen reduzieren.
